# UI原型设计规范

## 技术规范

### 单HTML多页面架构
- 所有页面代码放在同一个HTML文件中
- 使用JavaScript控制页面切换（隐藏/显示DOM元素）
- 页面容器使用`id`标识

### 样式规范
- 使用CSS实现所有视觉效果
- 禁止使用外部图片文件（使用SVG或CSS绘制）
- 定义全局CSS变量便于主题管理
- 响应式设计（媒体查询）

### 交互规范
- 所有按钮必须绑定点击事件
- 表单提交必须有验证和反馈
- 页面跳转必须有过渡动画
- 加载状态必须有视觉反馈

### 移动端适配
- 添加`viewport` meta标签
- 触摸事件支持（touchstart、touchend）
- 按钮最小点击区域44x44px
- 防止页面缩放和滚动穿透

## 视觉风格

### Modern风格
- 圆角设计（border-radius: 8px）
- 柔和阴影（box-shadow）
- 渐变色背景
- 卡片式布局

### Minimal风格
- 扁平化设计
- 大量留白
- 单色调配色
- 简洁的图标

### Colorful风格
- 鲜艳的配色
- 大胆的对比色
- 渐变色彩
- 活泼的动效

## 质量检查清单

- [ ] 所有按钮都能点击
- [ ] 页面切换流畅
- [ ] 表单验证完整
- [ ] 移动端显示正常
- [ ] 加载状态有反馈
- [ ] 无JavaScript错误
