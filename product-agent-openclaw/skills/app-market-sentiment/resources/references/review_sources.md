# 应用市场数据源配置

## 支持的应用市场

### iOS 平台
- **App Store**：Apple官方应用商店
  - 数据获取：App Store Connect API
  - 评论字段：评分、标题、内容、版本、日期、设备
  - 更新频率：实时

### Android 平台
- **华为应用市场**：华为官方应用商店
  - 数据获取：华为开发者联盟API
  - 评论字段：评分、内容、版本、日期、机型
  
- **小米应用商店**：小米官方应用商店
  - 数据获取：小米开放平台API
  - 评论字段：评分、标题、内容、版本、日期

- **应用宝**：腾讯应用商店
  - 数据获取：应用宝开发者平台
  - 评论字段：评分、内容、版本、日期

- **OPPO/vivo应用商店**
  - 数据获取：对应开发者平台

## 采集策略

### 增量监控
- 每日自动抓取新增评论
- 重点关注近7天和近30天数据
- 首次运行可抓取历史90天数据

### 版本关联
- 自动识别评论中的版本号（如"更新到v2.5后闪退"）
- 支持版本对比分析
- 标记版本发布后的评论变化

### 评分过滤
- 重点抓取1-3星低分评论（负面反馈）
- 抽样抓取4-5星评论中的建议部分（建设性意见）
- 1-2星评论100%采集，3星评论80%采集，4-5星评论20%抽样

## 数据格式

```json
{
  "reviews": [
    {
      "id": "review_001",
      "source": "appstore",
      "app_id": "com.example.app",

      "content": "更新后经常闪退，无法正常使用",
      "date": "2026-04-28T10:30:00",
      "device": "iPhone14,2",
      "language": "zh"
    }
  ]
}
```
