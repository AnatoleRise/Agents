# 图表类型选择指南

## 按数据类型选择

### 时间序列数据
- **折线图**：展示趋势变化（DAU趋势、容量变化）
- **面积图**：强调累积效果（累计用户数）
- **柱状图**：对比不同时间点的数值（日对比）

### 分类占比数据
- **饼图/环形图**：展示占比（情感分布、渠道分布）
- **堆叠柱状图**：展示构成变化（各渠道反馈量变化）
- **树图**：展示层级占比（问题分类层级）

### 对比数据
- **分组柱状图**：多维度对比（版本对比、渠道对比）
- **雷达图**：多指标综合对比（产品健康度评分）
- **箱线图**：数据分布对比（各版本评分分布）

### 关系数据
- **散点图**：展示相关性（用户vs容量）
- **热力图**：展示矩阵关系（时间×问题类型）
- **气泡图**：三维数据展示（用户×容量×满意度）

### 文本数据
- **词云**：展示高频关键词
- **情感河流图**：展示情感随时间变化
- **主题河流图**：展示话题热度变化

## 按场景选择

### 用户之声仪表盘
- 情感分布 → 饼图
- 痛点排行 → 水平柱状图
- 趋势变化 → 折线图
- 关键词 → 词云

### 核心功能数据看板
- 核心指标 → 数字卡片
- 趋势对比 → 折线图
- 量价关系 → 散点图
- 异常标记 → 带标记的折线图

## 配色方案

### 情感色彩
- 正面/积极：#52c41a（绿色）
- 中性/警告：#faad14（黄色）
- 负面/严重：#f5222d（红色）

### 品牌色彩
- 主色：#1890ff（蓝色）
- 辅助色：#722ed1（紫色）
- 强调色：#eb2f96（粉色）

### 数据序列
- 使用色板保证可区分性
- 避免色盲不友好的组合（红绿）
- 深色模式适配
