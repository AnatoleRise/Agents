# Difference Panel Template

## Purpose

Define standardized panels for product exploration reports. The panel should be concise enough for quick scanning and evidence-backed enough for decision review.

## Status Labels

| Label | Meaning |
|-------|---------|
| `领先` | Stronger or more complete evidence than competitors |
| `持平` | Similar evidence level compared with competitors |
| `缺失` | Evidence shows the capability or signal is absent |
| `未知` | Reliable information was not found in search/fetch results |

## Writing Rules

1. Every non-empty cell should include a status label and a citation.
2. Use "未知：未在搜索结果中找到" when evidence is missing.
3. Use `[unverified]` when evidence comes only from third-party summaries or weak sources.
4. Keep each cell concise: status + one short factual phrase + source.
5. The final "产品启示" column should identify what a product manager can learn or verify next.
