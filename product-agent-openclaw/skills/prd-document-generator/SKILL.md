# PRD文档生成技能

## 功能说明

根据需求描述生成标准化的PRD文档，支持PDF和DOCX两种格式，便于导入飞书在线编辑。

## 使用场景

- 产品需求规划
- 需求评审准备
- 开发指导文档
- 测试依据文档

## 输入参数

- `requirement`：需求描述（必需）
- `product_name`：产品名称（可选，默认"未命名产品"）
- `project_name`：项目名称（可选）
- `author`：作者姓名（可选）

## 输出内容

- PDF文档（24小时有效下载链接）
- DOCX文档（24小时有效下载链接）
- 文档内容预览

## 使用方式

主智能体调用时传递需求信息，子智能体直接生成文档并返回链接。

## 文档结构

按照PRD模板格式生成，包含以下章节：
- 修订记录
- 需求背景
- 需求目的
- 需求概述
- 功能需求
- 流程说明
- 非功能需求
- 风险
- 测试要点
